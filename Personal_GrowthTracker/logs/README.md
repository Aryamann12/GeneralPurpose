# Agent Logs Directory

This directory contains daily execution logs from all AI agents.

## Structure
```
logs/
├── Orchestrator_YYYY-MM-DD.md
├── FinanceAgent_YYYY-MM-DD.md
├── LearningAgent_YYYY-MM-DD.md
├── FocusAgent_YYYY-MM-DD.md
├── MoodAgent_YYYY-MM-DD.md
└── SchedulerAgent_YYYY-MM-DD.md
```

## Log Format

Each agent creates a daily log with:
- **Execution time** - When the agent ran
- **Inputs processed** - What data it analyzed
- **Outputs generated** - Actions taken or insights produced
- **Alerts/Warnings** - Important flags for user attention
- **Next steps** - Recommendations for tomorrow

## Retention Policy
- **Keep**: Last 90 days
- **Archive**: Older logs moved to `logs/archive/YYYY/MM/`
- **Summary**: Monthly summaries generated automatically

## Reading Logs

Logs are designed to be:
1. **Human-readable** - Clear markdown format
2. **Machine-parseable** - Structured data in JSON blocks
3. **Cross-referenced** - Links between related agent outputs

## Example Usage

To see today's complete picture:
1. Read `Orchestrator_<today>.md` for overview
2. Deep-dive into specific agent logs as needed
3. Check `SchedulerAgent_<today>.md` for tomorrow's plan

---

**Note**: These logs are auto-generated by agents. Manual edits may be overwritten.
